# Валенсия — персональный гид

Self-contained HTML-приложение для поездки 30.05–03.06.2026.

## Что внутри

- 91 место с описаниями, советами, ценами, часами
- Реальная карта (Leaflet + OpenStreetMap)
- Погода (Open-Meteo, обновляется автоматически)
- План по дням с дневником
- Контакты, транспорт, разговорник, блюда
- Работает оффлайн после первой загрузки

## Запуск

Просто открыть `index.html` в браузере. Лучше всего — захостить через GitHub Pages и открыть как PWA на телефоне (Share → На экран Домой).

## Технологии

- Pure HTML + vanilla JS, без сборки
- Leaflet 1.9.4 для карты
- Open-Meteo API для погоды
- LocalStorage для состояния
- Service Worker для оффлайн-кеша тайлов

## Обновление

Поменять `index.html` → `git push` → GitHub Pages автоматически передеплоит.
